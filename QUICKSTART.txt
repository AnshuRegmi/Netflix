## ğŸ”§ Troubleshooting

### Video Not Playing?
- Ensure browser allows autoplay
- Check internet connection
- Video should be muted by default

### Sliders Not Working?
- Check browser console (F12)
- Look for jQuery/Slick errors
- Refresh the page

### Modal Not Opening?
- Click "More Info" button
- Or click any content item
- Press ESC to close

### Styles Look Wrong?
- Clear browser cache (Ctrl+Shift+R)
- Check CSS files are loaded in DevTools
- Verify file paths are correct

### Nothing Works?
1. Open DevTools (F12)
2. Check Console tab for errors
3. Check Network tab - all files loaded?
4. Try different browser
5. Restart browser
